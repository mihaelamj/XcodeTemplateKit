# Nested Field Structure

Complete hierarchical field structure with examples.

## Structure

Each field path gets its own nested folder structure following the field hierarchy.

Example:
```
AssociatedTargetSpecification/
├── AllowableProductTypes/
│   ├── Framework_value.md
│   ├── Framework_complete.md
│   └── ...
├── PopUpTitle/
│   ├── Framework_value.md
│   └── Framework_complete.md
└── ...
```
