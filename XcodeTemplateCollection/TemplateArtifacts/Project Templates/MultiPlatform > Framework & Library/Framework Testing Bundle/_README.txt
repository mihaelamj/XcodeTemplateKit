Project Template: Framework Testing Bundle
Category: MultiPlatform > Framework & Library
Identifier: com.apple.dt.unit.multiPlatform.frameworkUnitTestBundle

This is a project template that requires Xcode to instantiate.

To generate a project from this template:
1. Open Xcode
2. File > New > Project
3. Navigate to the template category: MultiPlatform > Framework & Library
4. Select "Framework Testing Bundle"

Alternatively, use the AppleScript automation in generate_project.scpt
