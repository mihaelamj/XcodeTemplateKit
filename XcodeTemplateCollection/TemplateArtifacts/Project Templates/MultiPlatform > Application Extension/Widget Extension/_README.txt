Project Template: Widget Extension
Category: MultiPlatform > Application Extension
Identifier: com.apple.dt.unit.multiPlatform.widget

This is a project template that requires Xcode to instantiate.

To generate a project from this template:
1. Open Xcode
2. File > New > Project
3. Navigate to the template category: MultiPlatform > Application Extension
4. Select "Widget Extension"

Alternatively, use the AppleScript automation in generate_project.scpt
