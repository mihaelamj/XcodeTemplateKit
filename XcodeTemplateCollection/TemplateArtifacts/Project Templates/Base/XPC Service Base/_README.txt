Project Template: XPC Service Base
Category: Base
Identifier: com.apple.dt.unit.xpcservicebase

This is a project template that requires Xcode to instantiate.

To generate a project from this template:
1. Open Xcode
2. File > New > Project
3. Navigate to the template category: Base
4. Select "XPC Service Base"

Alternatively, use the AppleScript automation in generate_project.scpt
