Project Template: External Build System
Category: Other
Identifier: com.apple.dt.unit.externalBuildSystem

This is a project template that requires Xcode to instantiate.

To generate a project from this template:
1. Open Xcode
2. File > New > Project
3. Navigate to the template category: Other
4. Select "External Build System"

Alternatively, use the AppleScript automation in generate_project.scpt
