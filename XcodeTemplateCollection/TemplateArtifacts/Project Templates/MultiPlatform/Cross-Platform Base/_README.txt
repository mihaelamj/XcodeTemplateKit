Project Template: Cross-Platform Base
Category: MultiPlatform
Identifier: com.apple.dt.unit.crossPlatformBase

This is a project template that requires Xcode to instantiate.

To generate a project from this template:
1. Open Xcode
2. File > New > Project
3. Navigate to the template category: MultiPlatform
4. Select "Cross-Platform Base"

Alternatively, use the AppleScript automation in generate_project.scpt
